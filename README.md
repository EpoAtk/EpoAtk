# EpoAtk
The implementation of our ICDM 2020 paper "Exploratory Adversarial Attacks on Graph Neural Networks".
## Requirements
python == 3.6.8  
  torch == 1.1.0  
    numpy == 1.16  
     scipy == 1.2.1  
        networkx == 1.11
## Preprocess
unzip data.zip in ./src. 
## Attack
cd ./src.  
  zsh scripts with default hyper-parameters.
